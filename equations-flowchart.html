🔢 משוואה פשוטה (ללא סוגריים / מכנים)
1️⃣
העבר אברי x לצד שמאל
כשמעבירים — הסימן מתהפך!
5x − 2 = 2x + 7 → 5x − 2x = 7 + 2
2️⃣
צמצם אברים דומים
חבר/חסר את כל אברי x יחד, ואת כל המספרים יחד
3x = 9
3️⃣
חלק את שני הצדדים במקדם x
כדי לבודד את x לגמרי
x = 9 ÷ 3 = 3
4️⃣
✅ בדיקה — הצב במשוואה המקורית
שני הצדדים צריכים להיות שווים
5(3)−2 = 13 ✓   2(3)+7 = 13 ✓
🎉 x = 3 — זה הפתרון!
💡 כלל הזהב: מה שעושים לצד אחד — חייבים לעשות גם לצד השני!
🟠 משוואה עם סוגריים
1️⃣
פתח סוגריים — חוק הפילוג
המספר שמחוץ לסוגריים מוכפל בכל אבר בתוכם
3(x + 2) = 3x + 6
⚠️ שים לב לסימן מינוס לפני סוגריים!
מינוס הופך את כל הסימנים:   −(x − 3) = −x + 3
2️⃣
העבר x לשמאל, מספרים לימין
הסימן מתהפך בכל העברה
3x + 6 = x + 10 → 3x − x = 10 − 6
3️⃣
צמצם וחלק
2x = 4 → x = 2
4️⃣
✅ בדיקה
3(2+2)=12 ✓   2+10=12 ✓
🎉 x = 2 — זה הפתרון!
📝 דוגמה: 3(x + 2) = x + 10
פתח סוגריים:3x + 6 = x + 10
x לשמאל:3x − x = 10 − 6
צמצם:2x = 4
חלק:x = 2
בדיקה:3(4)=12 ✓   2+10=12 ✓
🟣 משוואה עם מכנים (שברים)
1️⃣
מצא מכנה משותף קטן ביותר (מ.מ.ק)
המספר הקטן שכל המכנים מתחלקים בו בשלמות
x/2 + x/3 = 5 → מ.מ.ק = 6
2️⃣
הכפל את כל המשוואה במ.מ.ק
כופלים כל אבר — ימין ושמאל — כדי לבטל את כל המכנים
×6:   3x + 2x = 30
⚠️ בדוק: אחרי הכפל, כל המכנים אמורים להיעלם!
אם נשאר מכנה — חזור וכפל שוב את כל האברים.
3️⃣
פתור את המשוואה הרגילה שנשארת
עכשיו אין יותר שברים — פתרון רגיל!
5x = 30 → x = 6
4️⃣
✅ בדיקה — הצב במשוואה המקורית עם השברים
6/2 + 6/3 = 3 + 2 = 5 ✓
🎉 x = 6 — זה הפתרון!
📝 דוגמה: x/4 + 1 = (x+2)/2
מ.מ.ק של 4 ו-2:4
הכפל ב-4:x + 4 = 2(x+2)
פתח סוגריים:x + 4 = 2x + 4
x לשמאל:x − 2x = 4 − 4
צמצם וחלק:−x = 0 → x = 0
בדיקה:0+1 = 1 ✓   2/2 = 1 ✓
💡 טיפ: אם המכנים שונים לגמרי — פשוט תכפול אותם זה בזה. למשל מכנים 3 ו-4 → מ.מ.ק = 12.

4

✅
בדיקה — הצב x ו-y בשתי המשוואות המקוריות
שתי המשוואות חייבות להתקיים בו-זמנית!
x=2,y=1:   2+1=3 ✓   2(2)+1=5 ✓
🎉 הפתרון: x = 2, y = 1
📝 דוגמה בהצבה: x + y = 3 , 2x + y = 5
בודד y:y = 3 − x (ממשוואה 1)
הצב במשוואה 2:2x + (3 − x) = 5
פשט:x + 3 = 5 → x = 2
מצא y:y = 3 − 2 = 1
בדיקה:2+1=3 ✓   4+1=5 ✓
