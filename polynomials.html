שאלות פרמטר — סוגים מהבגרות
⚠️ שאלת פרמטר = נותנים f(x) עם מספר לא ידוע a, ומידע נוסף. צריך למצוא את a.
סוג 1 — נתון שיש קיצון ב-x₀

דוגמה (קיץ תשפ"ה שאלה 7) g(x) = f(x) + a, קיצון פנימי ב-x = 5/2

סוג 2 — נתון שהגרף עובר בנקודה (x₀, y₀)
שיטההצב x₀ ו-y₀ ישירות בf(x) → פתור לa
דוגמה f(x) = x³ + ax, הגרף עובר ב-(2, 2)
2 = 8 + 2a → 2a = −6 → a = −3
סוג 3 — נתון שיפוע משיק ב-x₀
שיטהגזור → הצב x₀ → שווה לשיפוע הנתון → פתור לa
דוגמה f(x) = x³ − ax, שיפוע המשיק ב-x=1 הוא 0
f′(x) = 3x² − a → f′(1) = 3 − a = 0 → a = 3
