מינימום < k < מקסימום → 2 חיתוכים
k = מינימום → 1 או 2 — בדוק בכל מקרה לגופו

4. אם לא (חד-מונוטונית): אי-אפשר לחתוך פעמיים — כתוב "אין ערכי k כאלה"

🔢 פרמטרים — מצא את a
מה זה פרמטר בבגרות? נותנים לך פונקציה עם מספר לא ידוע (a) ומידע נוסף — ואתה צריך למצוא את a. המידע יכול להיות: נקודה שהגרף עובר בה, שיפוע משיק בנקודה, או מיקום נקודת קיצון.
סוג 1 — נתון שהגרף עובר בנקודה (x₀, y₀)
שיטה: הצב את x₀ ו-y₀ בפונקציה → פתור את המשוואה עבור a
דוגמה מבגרות (חורף תשפ״ה שאלה 6): f(x) = x/(x²+x+a), הגרף עובר ב-(2, 9.5/...)
הצב x=2 ואת ה-y הנתון → מצא a
סוג 2 — נתון שיפוע המשיק בנקודה x₀
שיטה: חשב f′(x) (עם a) → הצב x=x₀ → שווה לשיפוע הנתון → פתור
דוגמה מבגרות (חורף תשפ״ו שאלה 6): f(x) = a/(x−5), בנקודה x=7 שיפוע המשיק הוא −2
f′(x) = −a/(x−5)² → הצב x=7 → −a/4 = −2 → a = 8
סוג 3 — נתון שיש קיצון בנקודה x₀
שיטה: חשב f′(x) (עם a) → הצב x=x₀ → שווה ל-0 → פתור
דוגמה מבגרות (קיץ תשפ״ה שאלה 7): g(x) = f(x) + a, לפונקציה g יש מקסימום פנימי בנקודה x = 5/2
g′(x) = f′(x) → הצב x=5/2, שווה 0 → מצא a

תרשים זרימה — איך מתחילים שאלת פרמטר
קיבלת f(x) עם a ומידע נוסף?
↓
המידע הוא נקודה (x,y)? → הצב ישירות בפונקציה
המידע הוא שיפוע משיק? → גזור קודם, אז הצב

📈 גרף הנגזרת — איזה גרף מתאר את f′?
מה שואלים בבגרות? נותנים 4 גרפים (I–IV) ושואלים "איזה מתאר את f′(x)?". צריך לדעת לקרוא מה f′ אומרת על f ולהפך.
הכלל הבסיסי — קשר בין f ל-f′
f עולה ←→ f′ חיובית אם הגרף של f עולה באזור מסוים → f′ מעל לציר x באותו אזור
f יורדת ←→ f′ שלילית אם הגרף של f יורד באזור מסוים → f′ מתחת לציר x באותו אזור
קיצון של f ←→ אפס של f′ בנקודת מקסימום או מינימום של f → f′ חוצה את ציר x (f′=0)
נקודת קצה ←→ f′ לא מוגדרת שם בקצה התחום — f′ לא קיימת (הנגזרת לא מוגדרת בקצה)
אסטרטגיה — איך בוחרים את הגרף הנכון
שלב 1 — בדוק תחום f′ מוגדרת רק בתוך התחום הפתוח (לא בקצוות) → פסול גרפים עם ערכים מחוץ לתחום
שלב 2 — בדוק סימן היכן f עולה? שם f′ חייבת להיות מעל ציר x → פסול גרפים שלא עומדים בזה
שלב 3 — בדוק אפסים מספר האפסים של f′ = מספר הקיצונות הפנימיים של f
שלב 4 — בדוק גבולות פונקציית שורש: f′ מתקרבת לאינסוף ליד נקודת הקצה (המכנה שואף ל-0)
📝 דוגמה — f(x) = x·√(−x+4), תחום x≤4
f עולה ב:−√2 < x < √2 → f′ חיובית שם
f יורדת ב:x < −√2 ו-x > √2 → f′ שלילית שם
קיצונות:x=√2 (מקסימום) ו-x=−√2 (מינימום) → f′=0 בשתי נקודות
