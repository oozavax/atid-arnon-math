במשולש ישר זווית: a² + b² = c²
המשפט
ניסוח במשולש ישר זווית, סכום ריבועי הניצבים שווה לריבוע היתר.
סימונים a, b = הניצבים (הצלעות שיוצרות את הזווית הישרה)
c = היתר (הצלע הגדולה ביותר, מול הזווית הישרה)
הנוסחה a² + b² = c²
איך מזהים את היתר?
היתר הוא תמיד הצלע מול הזווית הישרה (90°)
היתר הוא תמיד הצלע הארוכה ביותר במשולש
אם הזווית הישרה ב-C — אז c = AB (הצלע שמחברת A ו-B)
כיוון הפוך — בדיקת משולש ישר זווית
המשפט ההפוך אם במשולש a² + b² = c², אז המשולש ישר זווית!
דוגמה: צלעות 3, 4, 5 3² + 4² = 9 + 16 = 25 = 5² ✓ → ישר זווית!
💡 טעות נפוצה: a² + b² = c² ולא a + b = c. תמיד ריבועים!
הוכחת משפט פיתגורס
הוכחה 1 — שטח ריבוע חיצוני (הוכחה גיאומטרית) ▼

רעיון: נבנה ריבוע עם צלע (a+b). בתוכו נשים 4 משולשים זהים ישרי זווית.

נתון
משולש ישר זווית עם ניצבים a, b ויתר c
בנייה
בנה ריבוע גדול עם צלע (a+b). בפינות הריבוע — 4 משולשים זהים ישרי זווית
שטח
שטח הריבוע הגדול = (a+b)² = a² + 2ab + b²
ניתוח
שטח הריבוע הגדול = שטח 4 המשולשים + שטח הריבוע הפנימי (צלע c)
= 4·(½ab) + c² = 2ab + c²
שוויון
a² + 2ab + b² = 2ab + c²
מסקנה
a² + b² = c² ✓
הוכחה 2 — שטחים (הוכחה אלגברית) ▼

רעיון: נשתמש בגובה מהזווית הישרה לשטח המשולש.

נתון
משולש ABC ישר זווית ב-C. נסמן: BC=a, AC=b, AB=c
גובה
נוריד גובה CH ל-AB. נסמן: AH=m, HB=n, CH=h
דמיון
משולש ACH דומה למשולש ABC → bc = mb → b² = mc
דמיון 2
משולש BCH דומה למשולש ABC → a² = nc
סיכום
a² + b² = nc + mc = c(n+m) = c·c = c² ✓
הוכחת ההפך — בדיקת ישר זווית ▼

טענה: אם a² + b² = c², אז המשולש ישר זווית.

נתון
משולש עם צלעות a, b, c כך ש-a² + b² = c²
בנייה
בנה משולש ישר זווית עם ניצבים a, b. יתרו הוא d, ולכן a² + b² = d²
השוואה
מהנתון: c² = a² + b² = d² → c = d
מסקנה
שני המשולשים קונגרואנטים (צ.צ.צ) → גם המשולש המקורי ישר זווית ✓
💡 לבגרות: בהוכחה — כל טענה צריכה נימוק! "נתון", "הגדרה", "משפט שלמדנו". אל תדלג על שלבים.
🔢 מחשבון פיתגורס

הכנס שתי צלעות ידועות, בחר מה לחשב:

דוגמאות פתורות
🟢 חישוב יתר — a=3, b=4 ▼
נוסחה
a² + b² = c²
הצב
3² + 4² = c² → 9 + 16 = c² → 25 = c²
פתרון
c = √25 = 5
🟢 חישוב ניצב — a=?, b=12, c=13 ▼
נוסחה
a² + b² = c²
הצב
a² + 144 = 169
פתרון
a² = 169 − 144 = 25 → a = 5
🟠 בעיית מילים — סולם על קיר (בינוני) ▼
שאלה: סולם באורך 10 מטר נשען על קיר. תחתית הסולם רחוקה 6 מ׳ מהקיר. באיזה גובה הסולם נוגע בקיר?

ציור
משולש ישר זווית: ניצב = גובה (h), ניצב = 6, יתר = 10
פיתגורס
h² + 6² = 10² → h² + 36 = 100
פתרון
h² = 64 → h = 8 מ׳
🔴 אלכסון מלבן + בדיקת ישר זווית (קשה) ▼
שאלה: מלבן עם אורך 8 ורוחב 6. א. מצא אלכסון. ב. האם משולש עם צלעות 7, 24, 25 ישר זווית?

א — אלכסון
d² = 8² + 6² = 64 + 36 = 100 → d = 10
ב — בדיקה
7² + 24² = 49 + 576 = 625 = 25² ✓ → כן, ישר זווית!

כפולות של שלשות
עיקרון אם (a,b,c) שלשה פיתגורית — אז גם (ka, kb, kc) לכל k חיובי!
דוגמה (3,4,5) → (6,8,10) → (9,12,15) → (12,16,20)...
משולשים מיוחדים
משולש שווה-שוקיים ישר זווית (45°-45°-90°) a = b → c = a√2
דוגמה: a=1, b=1, c=√2
משולש 30°-60°-90° אם הצלע הקטנה = a → הצלע האמצעית = a√3, היתר = 2a
💡 טיפ לבחינה: כשמקבלים מספרים כמו 3,4,? או 5,12,? — מיד בדוק אם זו שלשה פיתגורית מוכרת!
