÷ קיבלנו פונקציה רציונלית לחקור
1️⃣
תחום הגדרה
מצא לאילו ערכי x המכנה = 0 → אלה הערכים שאסורים
תחום = כל x פרט לערכים שמאפסים את המכנה
2️⃣
אסימפטוטות
אסימפטוטה אנכית: x = ערכים שמאפסים מכנה
אסימפטוטה אופקית: מה קורה כש-x → ∞ (לרוב y = 0 או y = מספר)
3️⃣
נקודות חיתוך עם הצירים
ציר x: f(x) = 0 → המונה = 0 (בדוק שהמכנה ≠ 0)
ציר y: הצב x = 0 (רק אם 0 בתחום!)
4️⃣
נגזרת וקיצון
גזור עם כלל המנה → f′(x) = 0 → נקודות קיצון
טבלת סימנים → קבע מקסימום / מינימום
קבע תחומי עלייה וירידה
5️⃣
סקיצה
סמן את האסימפטוטות כקווים מקווקוים
סמן חיתוכים וקיצונות
שים לב: הגרף לא חוצה אסימפטוטה אנכית!
✓ חקירה הושלמה
💡 הבדל מפולינום: ברציונלית — תמיד יש אסימפטוטה אנכית! זה הדבר הראשון שחייבים למצוא.
אסימפטוטות — מה זה ואיך מוצאים
אסימפטוטה אנכית (מאונכת לציר x)
מה זה?קו אנכי שהגרף מתקרב אליו אך לעולם לא מגיע
איך מוצאים?פתור: מכנה = 0 → התשובה היא x = ...
דוגמה: f(x) = 3x−5x − 5 = 0 → x = 5 → אסימפטוטה: x = 5
אסימפטוטה אופקית (מקבילה לציר x)
מה זה?קו אופקי שהגרף מתקרב אליו כש-x גדל מאוד
כשהמונה קבוע (למשל a/(x−b))כש-x → ∞, המחיר המונה קטן → f → 0 → y = 0
כשהמונה ומכנה אותה דרגההאסימפטוטה האופקית = מקדם מוביל של מונה ÷ מקדם מוביל של מכנה
דוגמה: f(x) = 2x+1x−3שניהם דרגה 1 → אסימפטוטה: y = 2/1 = 2
טיפוסי פונקציות רציונליות בבגרות
f(x) = ax−bאסימפטוטה אנכית: x=b · אסימפטוטה אופקית: y=0
f(x) = ax²+bxשתי אסימפטוטות אנכיות! · אסימפטוטה אופקית: y=0
f(x) = ax+bx+cאסימפטוטה אנכית: x=−c · אסימפטוטה אופקית: y=a
f(x) = x + axאסימפטוטה אנכית: x=0 · אין אסימפטוטה אופקית (יש אלכסונית)
💡 טיפ: בסקיצה — תמיד ציירי קודם את האסימפטוטות כקווים מקווקוים, ורק אחר כך את הגרף. זה עוזר לא לעשות טעויות!
כלל המנה
אם f(x) = u(x)v(x) אז f′(x) = u′·v − u·v′v²
איך זוכרים? — "מונה גורז מכנה, מכנה גורז מונה"
שלב 1זהה מי u (מונה) ומי v (מכנה)
שלב 2גזור כל אחד: u′ ו-v′
שלב 3הצב בנוסחה: (u′v − uv′) ÷ v²
שלב 4פשט את המונה אם אפשר
דוגמאות
דוגמה 1 — f(x) = 8x−5
u = 8, v = x−5u′ = 0, v′ = 1
f′(x)= 0·(x−5) − 8·1(x−5)² = −8(x−5)²
שים לב!נגזרת של ax−b תמיד שלילית (a>0) → פונקציה יורדת תמיד!
דוגמה 2 — f(x) = x²x²+1
u = x², v = x²+1u′ = 2x, v′ = 2x
f′(x)= 2x(x²+1) − x²·2x(x²+1)² = 2x³+2x−2x³(x²+1)² = 2x(x²+1)²
💡 טיפ: נגזרת של רציונלית — המכנה תמיד v², אף פעם לא שורש. אל תבלבל עם נגזרת שורש!
שאלות פרמטר — סוגים מהבגרות
⚠️ בכל בגרות יש לפחות סעיף אחד עם פרמטר ברציונלית. שלושת הסוגים:
סוג 1 — נתון שיפוע משיק ב-x₀
שיטהגזור עם כלל המנה → הצב x₀ → שווה לשיפוע הנתון → פתור
דוגמה (חורף 2026 שאלה 6) f(x) = ax−5, שיפוע משיק ב-x=7 הוא −2
f′(x) = −a(x−5)² → f′(7) = −a4 = −2 → a = 8
סוג 2 — נתון שהגרף עובר בנקודה (x₀, y₀)
שיטההצב x₀ ו-y₀ ישירות ב-f(x) → פתור
דוגמה (חורף תשפ"ה שאלה 6) f(x) = xx²+x+a, הגרף עובר ב-(2, 9.5/...)
הצב x=2 ואת ה-y הנתון → פתור ל-a
סוג 3 — נתון שיש קיצון ב-x₀
שיטהגזור → הצב x₀ → שווה 0 → פתור
דוגמה f(x) = x²+ax, יש מינימום ב-x=2
f′(x) = x²−ax² → f′(2) = 0 → 4−a=0 → a=4
💡 כלל: מידע על קיצון → גזור ואפס. מידע על נקודה → הצב ישירות. מידע על שיפוע → גזור והשווה.
תרגילים מבגרויות אמיתיות
🟢 חורף תשפ"ה 2025 — f(x) = x/(x²+x+a) (קל) ▼
שאלה 6 חורף תשפ"ה: f(x) = x/(x²+x+a), הגרף עובר ב-(2, 0.5). מצאו a.

הצב (2, 0.5)
0.5 = 2/(4+2+a) → 0.5(6+a) = 2 → 3+0.5a = 2 → a = −2

עבור a=−2: f(x) = x/(x²+x−2). מצאו תחום ואסימפטוטות.
תחום
x²+x−2 = 0 → (x+2)(x−1) = 0 → x≠−2 וx≠1
אסימפטוטות אנכיות
x = −2 ו-x = 1
אסימפטוטה אופקית
מונה דרגה 1, מכנה דרגה 2 → y = 0
🟢 קיץ תשפ"ה 2025 מועד ב׳ — f(x) = 12/(x²+4x) (קל-בינוני) ▼
שאלה 6 קיץ תשפ"ה מועד ב׳.

א. תחום, אסימפטוטות, חיתוך ציר x.
תחום
x²+4x=0 → x(x+4)=0 → x≠0 וx≠−4
אסימפטוטות אנכיות
x=0 ו-x=−4
אסימפטוטה אופקית
y=0 (מונה קבוע, מכנה דרגה 2)
חיתוך ציר x
מונה=12≠0 → אין חיתוך עם ציר x

ב. נגזרת וקיצון.
כלל המנה
f′ = (0·(x²+4x) − 12·(2x+4)) / (x²+4x)² = −12(2x+4)/(x²+4x)²
f′=0: מונה=0
2x+4=0 → x=−2 → f(−2)=12/(4−8)= −3 → מינימום (−2,−3)

ג. הישר y=k נפגש עם הגרף בנקודה אחת, 0<k. מצאו k.
רק קיצון מתאים
מינימום הוא הנקודה הנמוכה ביותר → k = −3 (אבל k>0, לכן בדוק מקסימום בתחום x>0)
🟠 קיץ תשפ"ה 2025 מועד א׳ — חקירה מלאה (בינוני) ▼
שאלה 6 קיץ תשפ"ה מועד א׳: f(x) = (x²+9x−½)/(x²+1)

א. אסימפטוטה אופקית.
שניהם דרגה 2
y = מקדם מוביל מונה ÷ מקדם מוביל מכנה = 1/1 = y=1

ב. נגזרת וקיצון.
u=x²+9x−½, v=x²+1
u′=2x+9, v′=2x
f′
= ((2x+9)(x²+1) − (x²+9x−½)·2x) / (x²+1)²
פשט מונה
2x³+2x+9x²+9 − 2x³−18x²+x = −9x²+3x+9 = −3(3x²−x−3)
f′=0
3x²−x−3=0 → x=(1±√37)/6
🔴 חורף תשפ"ו 2026 — פרמטר + חקירה מלאה (קשה) ▼
שאלה 6 חורף תשפ"ו: f(x) = a/(x−5), x≠5. בx=7 שיפוע המשיק הוא −2.

א. מצאו a.
נגזרת
f′(x) = −a/(x−5)²
הצב x=7
f′(7) = −a/(7−5)² = −a/4 = −2 → a = 8

עבור a=8: f(x) = 8/(x−5). חקירה מלאה.
תחום
כל x, פרט ל-x=5
אסימפטוטות
אנכית: x=5 · אופקית: y=0
חיתוך ציר y
f(0) = 8/(−5) = −1.6 → נקודה (0, −1.6)
חיתוך ציר x
מונה=8≠0 → אין
נגזרת
f′ = −8/(x−5)² < 0 תמיד → יורדת בכל תחום, אין קיצון
💡 טיפ לבגרות: כשמקבלים f(x) = a/(x−b) — הנגזרת תמיד שלילית (a>0). הפונקציה יורדת בכל תחום ואין קיצון. זו תשובה שלמה ולגיטימית!
